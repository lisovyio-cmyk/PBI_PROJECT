# PBI_PROJECT
## # PBI_PROJECT — Power BI Dashboard (GreenTech x Enedis)

Ce dépôt contient le projet Power BI et ses livrables (rapport + documentations).

## Contenu du dépôt

- **SAE_PBIX (1).pbix**  
  Rapport Power BI (tableau de bord complet : pages, visuels, mesures DAX, modèle).

- **Documentation_fonctionnelle_PowerBI_Greentech_Enedis.docx**  
  Documentation fonctionnelle (guide utilisateur) : comment naviguer dans le rapport, utiliser les filtres, comprendre les pages et les indicateurs.

- **Documentation_technique_PowerBI_Greentech_Enedis.docx**  
  Documentation technique : sources, préparation des données (Power Query), modèle de données (relations), mesures DAX, sécurité/RLS, performance.

- **README.md**  
  Présentation rapide du dépôt (ce fichier).

- **LICENSE**  
  Licence du dépôt (MIT).

## Ouvrir le projet

1. Télécharger **SAE_PBIX (1).pbix**
2. Ouvrir avec **Power BI Desktop**
3. Lire les documentations `.docx` si besoin (fonctionnelle / technique)
